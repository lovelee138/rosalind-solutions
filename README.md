# ROSALIND Solutions

This repository contains my independent solutions to problems from
[ROSALIND](https://rosalind.info/), a platform for learning bioinformatics
through problem solving.

Find more about my progress on telegram: https://t.me/sunflower_distribution

## Goals

- Learn the fundamentals of bioinformatics
- Improve my Python programming skills
- Practice working with biological sequences
- Study common algorithms used in computational biology
- Connect biological concepts with computational problems
